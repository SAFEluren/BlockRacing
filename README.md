# BlockRacing

Fork自 ： [LQSnow/BlockRacing](https://github.com/LQSnow/BlockRacing) 

基础功能见原仓库 在原仓库的基础上升级到1.20.1 增加了/brdebug 命令 用于op调试

开发环境依赖于Paper 因此不保证Paper上游（CraftBukkit/Spigot）可以正常使用
不受理非Paper分支的服务器兼容性问题
